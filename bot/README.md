## 🚀 التشغيل

### 1) المتطلبات
- Python 3.10+
- Redis (محلي : `redis-server`)
- ffmpeg (للتحميل)

### 2) أول تشغيل (يطلب البيانات تفاعلياً)
```bash
cd bot
chmod +x run.sh
./run.sh
```
سيُطلب منك :
- `API_ID` و `API_HASH` من [my.telegram.org](https://my.telegram.org)
- `BOT_TOKEN` من [@BotFather](https://t.me/BotFather)
- `Dev_Zaid` ايدي المطور

تُحفَظ في `settings.json` ولن تُطلب لاحقاً.

### 3) تشغيل بدون السكربت
```bash
pip install -r requirements.txt
python main.py
```

### 4) تنظيف
```bash
python clean.py             # أساسي
python clean.py --sessions  # + session
python clean.py --all       # شامل (Redis + ملفات)
```

## ⚠️ ملاحظات

- البوت **مفعّل افتراضياً** عند الإضافة لأي مجموعة.
